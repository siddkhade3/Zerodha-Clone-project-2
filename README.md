# Zerodha Clone

A Zerodha-inspired trading platform clone built to understand how real-world fintech applications work. This project focuses on authentication, stock trading flow, and frontend–backend integration.



## 🛠️ Technologies Used

* **Frontend:** React, Bootstrap 
* **Backend:** Node.js, Express
* **Database:** MongoDB, Mongoose
* Fetch data : axios 
* **API Testing:** Postman
* **Tools:** Git, GitHub

---

## ✨ Features

* User authentication
* Buy and sell stocks
* View holdings
* Order history tracking
* Clean and responsive landing page


## 🚀 How It Can Be Improved

* Add **real-time stock prices** using a live stock market API
* Improve UI spacing, charts, and responsiveness
* Add advanced order types (limit, stop-loss)
* Better real-time updates for holdings and orders

---

## ▶️ How to Run the Project

Start the server:

```bash
npm run start
```

Run in development mode:

```bash
npm run dev
```
